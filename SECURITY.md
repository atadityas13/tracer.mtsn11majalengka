# Security Policy

## Supported Versions

Versi yang didukung adalah branch aktif yang sedang dipakai operasional.

## Reporting a Vulnerability

Jika Anda menemukan celah keamanan, **jangan buat issue publik**.

Silakan laporkan melalui:

- Email: **mtsn11majalengka@gmail.com**
- Subjek: `Security Report - TRACER`

## Informasi Laporan Minimum

Mohon sertakan:

1. Ringkasan kerentanan
2. Lokasi file/fitur terdampak
3. Langkah reproduksi
4. Dampak potensial
5. Saran mitigasi (jika ada)

## Response Timeline

- Konfirmasi laporan: maksimal 3 hari kerja
- Validasi awal: maksimal 7 hari kerja
- Patch/perbaikan: menyesuaikan tingkat risiko

## Disclosure

Mohon tidak melakukan publikasi sebelum perbaikan dirilis.
